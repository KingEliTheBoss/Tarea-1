Etiqueta header: esta etiqueta define la cabecera de la pagina web, es decir que se agrupan cosas como el nombre de la pagina o tabla de contenidos

Etiqueta div: como la etiqueta header, esta tambien define una seccion de la pagina, ayudar a dar forma a la pagina

Etiquetas h1,h2,h3,h4,h5,h6: estas etiquetas son los niveles de importancia de texto en la cabecera, h1 es el de mayor importancia, y h6 es el de menor, se utiliza para dirigir la atencion del usuario

Etiqueta br: esta etiqueta funciona para hacer saltos de linea y separar mas eficientemente las agrupaciones
